Yodlr Front End Engineer Code/Design Challenge
=======================
Hello!

This project has been modified to use Vite.  To install and start the project:

```
npm i
cd server npm i
cd ..
npm start
```

To run the React and Express server separately:

React Server:
```
npm run dev
```

Express Server:
```
npm run server
```
